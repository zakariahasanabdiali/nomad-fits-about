Nomad Fits - Men's Clothing eCommerce

This project is a modern eCommerce website for men's clothing built using the Platzi Fake Store API. It demonstrates how to connect a frontend interface with a real API to display products dynamically. The focus is on building a responsive, user-friendly shopping platform suitable for portfolios and real-world learning.
